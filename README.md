# Homeshare Bite App

**Description:**  
Homeshare Bite is a mobile application built using **Flutter** and **Dart**. The app allows users to log in, navigate through a menu, and share content via the menu sharing screen.

**Features:**  
- Login screen  
- Main menu  
- Menu share screen  

**Technologies Used:**  
- Flutter  
- Dart  

**About Me:**  
This project was developed as part of my studies to gain hands-on experience in cross-platform app development, UI design, and Flutter programming.
