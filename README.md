# Homeshare Bite App

**Description:**  
Homeshare Bite is a mobile application built using **Flutter** and **Dart**. The app allows users to log in, navigate through a menu, and share content via the menu sharing screen.

**Features:**  
- Login screen  
- Main menu  
- Menu share screen  

**Technologies Used:**  
- Flutter  
- Dart  

**How to Run:**  
1. Clone the repository: `git clone https://github.com/SYAAFFIIQ/HOMESHARE-BITE-APP.git`  
2. Open the project in **VS Code** or **Android Studio**  
3. Run `flutter pub get` to install dependencies  
4. Run the app on an emulator or physical device
